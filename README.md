# 2-DOF-rolling-bearing-dynamics-model 二自由度（2-DOF）轴承动力学模型 matlab 数字孪生
基于赫兹接触理论  采用四阶龙格-库塔算法求解 包含XY轴的振动位移，振动速度，振动加速度，加速度频谱
在Sunnersjo动力学模型基础上建立的二自由度轴承模型
有复现的文献和详细的参数注释 可按自己需要的参数修改
